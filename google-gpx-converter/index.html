     Supported URL Formats
    
                    
    We support various Google Maps URL formats
    
                
    
                
      
                    
    •  Short URLs: https://maps.app.goo.gl/xyz123
      • 
                    
    •  Coordinates: https://www.google.com/maps/@40.7128,-74.0060,15z
      • 
                    
    •  Place queries: https://maps.google.com/maps?q=New+York
      • 
                    
    •  Directions: https://www.google.com/maps/dir/Start/End
      • 
                
    
                
                
    
                    
     Short URL Help
    
                    
    If you have a short URL like maps.app.goo.gl/xyz123:
    
                    
    
                        
    
                            
    1
    
                            
    Open the short URL in your browser
    
                        
    
                        
    
                            
    2
    
                            
    Wait for it to redirect to the full Google Maps page
    
                        
    
                        
    
                            
    3
    
                            
    Copy the full URL from your browser's address bar
    
                        
    
                        
    
                            
    4
    
                            
    Paste that expanded URL here instead
    
                        
    
                    
    
                    
                    
                    
     Getting Coordinate URLs
    
                    
    For place names, get coordinate URLs by:
    
                    
    
                        
    
                            
    1
    
                            
    Find the location on Google Maps
    
                        
    
                        
    
                            
    2
    
                            
    Right-click on the exact spot
    
                        
    
                        
    
                            
    3
    
                            
    Select "What's here?" from the menu
    
                        
    
                        
    
                            
    4
    
                            
    Copy the URL - it will contain @lat,lng coordinates

     About GPX
    
                
    GPX (GPS Exchange Format) is a data format for exchanging GPS data between applications and devices.
