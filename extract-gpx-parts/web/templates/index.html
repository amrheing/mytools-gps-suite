About GPX Parts Extractor

This tool extracts and organizes components from your GPX files into separate downloadable files:

  • Waypoints: All waypoints extracted into a single file
  • Tracks: Each track separated into an individual file, named from the track name in the GPX
  • Routes: Each route separated into an individual file

Uploaded files are kept in the archive. Re-uploading the same file (same build date) skips reprocessing and shows the existing results directly. Pressing ▶ on an archived file also reuses existing results without reprocessing.

Files can be deleted from the archive using the delete token.
