Welcome to My Tool Suite

A collection of specialized tools designed to make working with GPS data, web content, and file processing easier and more efficient.

Google Maps to GPX Converter

Convert Google Maps URLs to GPX files for use with GPS devices and navigation apps.

  • Convert multiple URLs at once
  • Support for various Google Maps formats
  • Customizable route names and timestamps
  • Auto URL expansion for short links
  • Instant preview and download

GPX to KML Converter

Convert GPX files to KML format for Google Earth and mapping applications.

  • Drag & drop file upload
  • Waypoints, tracks, and routes support
  • Configurable styling options
  • Real-time preview before download
  • Google Earth compatible output

GPX Parts Extractor

Extract and organize waypoints, tracks, and routes from your GPX files into separate components.

  • Extract waypoints, tracks, and routes
  • Organize into separate files
  • Batch download capabilities
  • Support for large GPX files (100MB+)
  • Secure processing with auto-cleanup

POI Pattern Extractor

Filter and extract Points of Interest from GPX files using wildcard or regex patterns.

  • Wildcard and regex pattern matching
  • Extract specific POI categories
  • Filter by naming conventions
  • Generate focused GPX files
  • Supports large files (500MB+)

Route Tracker

Real-time GPS route tracking with user management, route history, and comprehensive data export.

  • Real-time GPS tracking with live maps
  • User accounts and route history
  • Multiple export formats (GPX, KML, JSON, CSV)
  • Offline mode and mobile optimization
  • Route analytics and statistics
